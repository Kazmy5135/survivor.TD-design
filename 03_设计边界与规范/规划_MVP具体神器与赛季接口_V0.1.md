## 1. 文档目标

本文定义 MVP 首发神器池、每件神器的单件价值、支持流派、规则改变、事件边界、冲突关系和赛季扩展接口。

本节点确定神器“改变什么规则、怎样形成联动”，不填写最终伤害倍率、层数、目标数、冷却和费用返还数值。所有平衡数字保留到 D3；机制不可缺失的数字标记为机制上限或候选。

## 2. 神器总原则

### 2.1 首发池结构

首发提案共 18 件神器：

| 类别 | 数量 | 职责 |
|---|---:|---|
| 火属性神器 | 5 | 燃烧覆盖、复燃、爆破、技能轮转、击杀回收 |
| 冰属性神器 | 5 | 长期寒冷规则改写、追击、碎冰、冰锚、阵型分流 |
| 电属性神器 | 5 | 感电保留、群体放电、单体过载、动作打断、余压转移 |
| 通用桥接神器 | 3 | 异技能轮转、状态结算接力、控制回收 |

### 2.2 单件与组合边界

- 每件神器必须在只持有自身时产生完整且可理解的收益。
- 神器不能成为任何基础 Build 成立的唯一前提。
- 两件神器联动属于胡牌提升，三件完整吻合属于大胡；不设计“两件指定神器缺一不可”的基础配方。
- 神器不升级、不共鸣、不觉醒；强度来自规则匹配，而不是同件重复成长。
- 神器主要改变目标选择、状态结算、技能顺序、战场几何和费用节奏，不承担高频基础属性成长。

### 2.3 规则改写边界

任何规则改写神器都必须说明：

1. 被替换的基础规则；
2. 持有神器时的替代规则；
3. 未持有神器时仍采用的基础规则；
4. 原规则事件是否继续发出，监听者怎样迁移；
5. 获得神器时已有状态怎样处理；
6. 与其他神器的兼容、互斥和结算优先级。

神器不得覆盖 D1 的 `FORBIDDEN` 禁区，尤其不得让寒冷减速随层数提高。

### 2.4 通用触发安全

除单件另有说明外，所有神器统一遵守：

- 同一神器每个 `RootEventId` 最多启动一次；
- 神器产生的触发伤害、状态、控制和费用事件不能再次触发自身；
- 最大额外触发深度为 1；
- 同一目标间隔、单次释放次数、单位时间次数和真实目标数均为 `[待 D3]`，但配置字段必须存在；
- 死亡目标不再作为受击对象；需要保留位置效果时只使用死亡位置；
- 视觉目标数可以低于真实结算上限，但不能显示不存在的命中或控制。

## 4. 火属性神器

### 4.1 F-A1 燎原共鸣

一句话：一个目标首次达到燃烧满层时，向周围发出一次余焰，使燃烧从单点变成可维持的战场覆盖。

| 字段 | 定义 |
|---|---|
| 类型 | 公共输入、覆盖桥接 |
| 主要支持 | F1；次要支持 F2、F3 |
| 单件价值 | 任何能推进燃烧满层的技能都能把单体挂层转成有限范围覆盖 |
| 触发事件 | 非神器来源使目标从未满层跨到燃烧满层 |
| 作用对象 | 满层目标附近的合法敌人 |
| 效果 | 已燃烧目标刷新燃烧持续时间；未燃烧目标获得少量燃烧 |
| 结算时机 | 满层事件完成后加入神器队列 |
| 上限 | 每个根事件一次；单次释放目标上限与同目标间隔 `[待 D3]` |
| 防递归 | 本神器施加或刷新燃烧产生的满层事件不能再次触发燎原共鸣 |
| 死亡处理 | 满层目标若同时死亡，以死亡位置发出余焰，不再以其为受击目标 |
| 信息反馈 | 满层目标出现一次向外扩张的火环；刷新与新施加使用不同亮度，不显示精确层数 |

### 4.2 F-A2 余烬复燃

一句话：主动清空燃烧不会让循环完全断掉，而是在目标或死亡位置留下一个必须由“另一技能”接起的可见余烬。

| 字段 | 定义 |
|---|---|
| 类型 | 结算后重启、异技能桥接 |
| 主要支持 | F2；次要支持 F1 |
| 单件价值 | 任何全量清除燃烧的技能都获得下一轮燃烧的明确重启点 |
| 触发事件 | 主动技能全量清除目标当前燃烧 |
| 作用对象 | 存活目标；若目标死亡，则作用于死亡位置 |
| 效果 | 留下可见余烬印记；下一次由不同主动技能造成的合法命中点燃余烬并施加少量燃烧 |
| 同事件边界 | 清除燃烧的同一技能、同一命中和其触发伤害不能点燃余烬 |
| 上限 | 每目标最多一个余烬；全场数量、持续时间和同目标间隔 `[待 D3]` |
| 防递归 | 余烬施加的燃烧不能生成新的余烬，也不能由神器伤害直接点燃 |
| 死亡处理 | 地面余烬由第一个进入并被下一不同技能合法命中的敌人承接；超时消失 |
| 信息反馈 | 清除后留下暗红余烬；可点燃时脉动，成功接力后明亮复燃 |

### 4.3 F-A3 爆燃震线

一句话：燃爆不只造成局部伤害，还沿远离墙体的方向推出一条火焰震线，把爆发转成防线空间。

| 字段 | 定义 |
|---|---|
| 类型 | 爆破收口、伤害转控制 |
| 主要支持 | F2；可桥接 I3、E3 的墙前管理 |
| 单件价值 | 任何燃烧全量清除产生的燃爆都附带一次阵型整理 |
| 触发事件 | 燃烧被全量清除并成功产生燃爆 |
| 作用对象 | 爆燃中心至远离墙体方向带状区域内的合法敌人 |
| 效果 | 造成一次触发伤害并把目标轻度推离墙体 |
| 上限 | 每次燃爆最多一条震线；每目标每根事件最多命中一次；目标数 `[待 D3]` |
| 防递归 | 震线不施加燃烧、不清除燃烧、不产生新燃爆，不能再次触发本神器 |
| Boss 边界 | Boss 保留触发伤害，击退按控制抗性降低或免疫 |
| 信息反馈 | 爆燃中心向上推出一条窄而清晰的地表火线，与普通范围爆炸区分 |

### 4.4 F-A4 炉心换挡

一句话：玩家用不同主动技能轮流处理燃烧时，费用循环变得更顺；连续重复同一技能则没有收益。

| 字段 | 定义 |
|---|---|
| 类型 | 供能、操作顺序改变 |
| 主要支持 | F3；次要支持 F1、F2 |
| 单件价值 | 只要持有一个燃烧技能和另一个可命中技能，就能通过交替释放获得费用回流 |
| 触发条件 | 本次主动技能与上一次完成释放的主动技能不同，且本次成功施加、刷新或全量清除燃烧 |
| 结算时机 | 本次释放结束后返还部分实际支付费用 |
| 上限 | 每次释放最多一次；单位时间上限 `[待 D3]` |
| 费用边界 | 返还必定低于本次实际支付费用；费用不足而未成功释放时不触发 |
| 防递归 | 自动重复仍按实际技能 ID 判断；神器、被动和触发伤害不作为“另一主动技能” |
| 信息反馈 | 可获得换挡收益的技能按钮出现炉心连线；返还时费用池出现一次回流脉冲 |

### 4.5 F-A5 灰烬收成

一句话：燃烧持续伤害完成击杀时，对应技能能量点不再等待三秒，而是立刻被回收进费用池。

| 字段 | 定义 |
|---|---|
| 类型 | 击杀回收、资源节奏改变 |
| 主要支持 | F3；次要支持 F1 |
| 单件价值 | 燃烧击杀立即加快下一次技能释放，不需要另一件神器配合 |
| 触发事件 | 燃烧持续伤害造成合法击杀，且该死亡实际生成技能能量点 |
| 效果 | 对应技能能量点立即执行拾取，并应用神器专属恢复修正 `[待 D3]` |
| 上限 | 单位时间立即拾取次数 `[待 D3]`；一次死亡只处理一次 |
| 资源边界 | 不额外制造掉落；没有技能能量点掉落时不凭空恢复费用；升级经验仍遵守原拾取规则 |
| 防递归 | 即时拾取产生的费用事件不能再次触发神器或复制掉落 |
| 信息反馈 | 燃烧击杀位置的能量点化为一条灰烬流立即飞向费用池 |

## 5. 冰属性神器

### 5.1 I-A1 无尽寒域

一句话：放弃冰冻，把寒冷改造成可以长期堆高、持续追击的深寒状态。

| 字段 | 定义 |
|---|---|
| 类型 | 核心规则改写、I1 收口 |
| 主要支持 | I1 极寒追击 |
| 单件价值 | 一件神器同时消除频繁冰冻空窗，并给高层寒冷建立连续追击出口 |
| 被替换基础规则 | 寒冷达到基础上限时全部转化为冰冻 |
| 替代规则 | 寒冷达到神器上限后保留在目标身上，不转化冰冻；寒冷上限改为 `[候选：10 层]` |
| 额外施加 | 玩家主动技能成功施加寒冷时额外增加一层；神器触发与持续伤害不获得该额外层 |
| 输出出口 | 玩家主动技能对寒冷目标造成直接伤害时，按当前寒冷层数获得连续追击修正 `[待 D3]`；不存在中间层数阈值 |
| 减速禁区 | 从 1 层到神器上限始终使用同一个固定减速，绝不随层数提高 |
| 获得时迁移 | 已存在寒冷层数原样保留并改用新上限；已经冰冻的目标保持当前冰冻，冰冻结束后才使用新规则 |
| 事件迁移 | 到达神器上限仍发出一次“寒冷满层”事件，但不发出“进入冰冻”事件 |
| 互斥 | 与依赖冰冻、碎冰、冰冻锚点的 I-A3、I-A4、I-A5 互斥，不在同局后续神器选择中共同出现 |
| 信息反馈 | 满层后进入深寒外观，霜层更厚但移动减速表现不继续增强；不显示精确层数 |

### 5.2 I-A2 霜猎折返

一句话：不同主动技能接力命中寒冷目标时，产生一次沿原轨迹返回的追击碎片。

| 字段 | 定义 |
|---|---|
| 类型 | 异技能追击、持续输出桥接 |
| 主要支持 | I1；次要支持 I2 |
| 单件价值 | 一个寒冷输入技能加任意另一命中技能即可形成额外追击 |
| 触发条件 | 当前主动技能与最近一次给目标施加或刷新寒冷的主动技能不同，并对该寒冷目标造成直接伤害 |
| 效果 | 短延迟后生成一枚折返碎片，追击原目标；原目标死亡或不再寒冷时改锁最近寒冷目标 |
| 伤害类型 | 碎片为触发伤害，不施加寒冷，不读取冰冻目标作为有效追击条件 |
| 上限 | 每目标间隔、每次释放次数与全场碎片数 `[待 D3]` |
| 防递归 | 折返碎片不能触发霜猎折返，也不能触发依赖主动直接命中的同类效果 |
| 信息反馈 | 两个技能之间出现一条短暂霜线，碎片沿霜线折返追击 |

### 5.3 I-A3 借时碎晶

一句话：提前碎冰时，目标尚未使用完的冰冻时间不被浪费，而是立即转换成碎晶规模。

| 字段 | 定义 |
|---|---|
| 类型 | 控制时间换爆发、I2 收口 |
| 主要支持 | I2 碎冰爆发 |
| 单件价值 | 任意碎冰技能都能根据玩家牺牲的剩余控制时间获得更明确的爆发回报 |
| 触发事件 | 玩家主动技能提前结束目标冰冻并成功产生碎冰 |
| 效果 | 按碎冰前剩余冰冻时间连续计算碎晶总量、穿透或搜索范围 `[待 D3]` |
| 作用对象 | 周围合法非死亡目标；优先未冰冻目标 |
| 状态结果 | 碎晶造成触发伤害后可施加少量寒冷 `[待 D3]`，但不能直接执行碎冰 |
| 上限 | 每次碎冰启动一次；真实碎晶数、目标数和视觉合并规则 `[待 D3]` |
| 防递归 | 碎晶伤害和其施加的寒冷不能再次触发借时碎晶；同根事件最大深度 1 |
| 信息反馈 | 冰冻剩余越多，碎裂时形成的晶体体量或亮度越明显，不要求玩家读取时间数字 |

### 5.4 I-A4 冰锚连城

一句话：冰冻目标不再是孤立控制点；两个可见冰锚之间会拉出一次寒潮连线，把中间敌群推离墙体。

| 字段 | 定义 |
|---|---|
| 类型 | 冰冻锚点网络、群体控场 |
| 主要支持 | I3；次要支持 I2 |
| 单件价值 | 第一个冰锚也会产生一次较弱局部回拒；出现第二个冰锚后升级为连线控制 |
| 触发事件 | 目标进入冰冻并成为新的冰锚 |
| 效果 | 若存在前一个有效冰锚，在两锚之间生成一次寒潮连线，对线段附近敌人造成触发伤害并推离墙体 |
| 锚点规则 | 全场最多保留最近两个可见锚点；锚点只是位置记录，不是可受击单位 |
| 上限 | 每个新冰锚最多生成一条连线；每目标每根事件最多结算一次；宽度和目标数 `[待 D3]` |
| 防递归 | 连线伤害不施加寒冷、不制造新冰锚、不再次触发连城 |
| Boss 边界 | Boss 保留伤害，位移按控制抗性降低或免疫 |
| 信息反馈 | 两个冰锚之间出现清晰霜桥，寒潮沿桥向远离墙体方向扫过 |

### 5.5 I-A5 回拒棱镜

一句话：冰冻产生的回拒被折射成左、中、右三路寒潮，把敌群推远的同时重新压整成可被范围技能利用的队形。

| 字段 | 定义 |
|---|---|
| 类型 | 控制几何改写、阵型分流 |
| 主要支持 | I3；辅助 F2、E1 等范围输出 |
| 单件价值 | 即使没有寒潮封锁专属技能，任意冰冻事件也会产生一次较弱的三路回拒 |
| 触发事件 | 目标进入冰冻 |
| 替代规则 | 若该事件已经会产生寒潮回拒，则把原径向回拒替换为左、中、右三条向远离墙体推进的窄波；不额外复制第四次回拒 |
| 效果 | 每个合法目标只进入一条波，被推离墙体并向对应波中心轻度横向整理 |
| 上限 | 三条波共享同一个真实目标预算；每目标每根事件一次；范围与目标数 `[待 D3]` |
| 寒冷边界 | 不增加寒冷层数，不改变固定减速，不延长冰冻 |
| Boss 边界 | Boss 可以免疫横向整理和击退，但不取消已有技能基础伤害 |
| 信息反馈 | 冰锚前方出现三棱折射光，随后三条寒潮带并行推出 |

## 6. 电属性神器

### 6.1 E-A1 闭合回路

一句话：电击完成后不再清空感电，使电从“充一次、放一次”变成受间隔限制的持续闭路放电。

| 字段 | 定义 |
|---|---|
| 类型 | 核心规则改写、持续放电 |
| 主要支持 | E1、E2、E3 |
| 单件价值 | 任意感电来源加任意直接命中来源即可持续触发电击，不需要重新播种 |
| 被替换基础规则 | 电击结算后清除目标当前全部感电 |
| 替代规则 | 电击结算后保留触发前的感电层数；同一次命中仍不能重挂或追加感电 |
| 触发边界 | 每目标存在电击最小间隔 `[待 D3]`；每个根事件最多触发一次电击 |
| 事件迁移 | 每次电击后仍发出“感电结算完成”和“本应清除”事件，供其他神器读取，但不发出真实“感电已清除”事件 |
| 获得时迁移 | 当前感电层数原样保留；不立即触发电击，也不补发历史事件 |
| 防递归 | 电击和神器触发伤害不能再次触发感电；直接命中才可启动下一次合法电击 |
| 信息反馈 | 电击后感电标记不消失，而是短暂闭合成环；最小间隔内标记处于冷却暗态 |

### 6.2 E-A2 群星导线

一句话：一次主动放电会沿可见导线顺序结算附近多个已感电目标，而不是只处理被直接命中的那个目标。

| 字段 | 定义 |
|---|---|
| 类型 | 群体放电、E1 收口 |
| 主要支持 | E1 弧链放电 |
| 单件价值 | 多目标感电覆盖可以直接转成一次有限群体结算 |
| 触发事件 | 一次主动技能直接命中成功触发本根事件的第一次电击 |
| 作用对象 | 原目标附近已感电且尚未被本根事件结算的目标 |
| 效果 | 按距离或邻接顺序依次触发各目标自己的感电电击；默认结算后各自清除感电 |
| 闭路兼容 | 持有 E-A1 时，各目标改为保留感电，但仍受每目标间隔和每根事件一次限制 |
| 上限 | 每根事件启动一次导线；最大附加目标数和搜索距离 `[待 D3]` |
| 防递归 | 导线触发的后续电击不能再次启动群星导线，也不能把同一目标加入两次 |
| 信息反馈 | 多个感电目标之间先显示细导线，再按稳定顺序逐个脉冲，避免同时爆闪 |

### 6.3 E-A3 过载避雷针

一句话：达到感电满层的目标成为一个清晰可见的避雷针，下一次直接命中会得到额外的单体聚焦收口。

| 字段 | 定义 |
|---|---|
| 类型 | 满层标记、单体过载 |
| 主要支持 | E2；次要支持 E3 |
| 单件价值 | 任意能把目标推进感电满层的技能都能建立一次明确的高价值命中窗口 |
| 触发事件 | 目标从未满层跨到感电满层 |
| 效果 | 目标获得可见避雷针标记；下一次玩家主动直接命中除正常电击外，再产生一次按触发前层数计算的聚焦电击 `[待 D3]` |
| 消耗 | 聚焦电击后移除避雷针标记；基础感电按当前规则清除或由 E-A1 保留 |
| 再次建立 | 只有目标离开满层并再次跨入满层时才能重新获得标记；满层刷新不重复触发 |

| 防递归 | 聚焦电击为触发伤害，不触发感电、避雷针或其他直接命中监听 |
| 信息反馈 | 满层目标上方形成明显避雷针轮廓；被命中时雷柱向目标集中收束 |

### 6.4 E-A4 断招回授

一句话：对正在准备危险动作的目标放电时，不只打断该目标，还释放一次不造成伤害的群体动作延迟脉冲。

| 字段 | 定义 |
|---|---|
| 类型 | 威胁打断、控制扩散 |
| 主要支持 | E3；次要支持 E2 |
| 单件价值 | 任意感电加直接命中组合都能针对可观察危险动作提供一次控制收益 |
| 触发条件 | 电击结算时目标正处于可中断的攻击、冲锋、蓄力或施法准备阶段 |
| 效果 | 中断原目标动作；若中断成功，向附近敌人释放一次动作延迟脉冲，不造成伤害 |
| 作用对象 | 附近当前正在执行可延迟动作的合法敌人 |
| 上限 | 每目标间隔、每次释放成功中断数、脉冲目标数 `[待 D3]` |
| Boss 边界 | Boss 可以只受到动作进度削减或完全免疫控制；电击伤害仍正常结算 |
| 防递归 | 延迟脉冲不能触发电击、感电施加或新的断招回授 |
| 信息反馈 | 原目标动作特效被切断，随后出现一次无伤害的蓝白节拍波；受影响目标动作条后退或停顿 |

### 6.5 E-A5 余压转接

一句话：感电完成一次结算后，把一层余压送到另一名尚未感电的敌人，让下一轮播种从战场另一点继续。

| 字段 | 定义 |
|---|---|
| 类型 | 结算后转场、群体播种 |
| 主要支持 | E1、E3 |
| 单件价值 | 每次放电都能为另一目标建立下一轮感电入口 |
| 触发事件 | 感电电击结算完成；持有 E-A1 时读取“本应清除”事件 |
| 作用对象 | 距离原目标最近的未感电合法敌人，不得选择原目标 |
| 效果 | 以状态事件施加一层感电，不造成伤害 |
| 上限 | 每根事件最多转接一次；搜索距离与单位时间次数 `[待 D3]` |
| 防递归 | 余压施加不触发电击、不触发余压转接、不启动群星导线；后续真实直接命中可以正常放电 |
| 死亡处理 | 原目标死亡时从死亡位置搜索；没有合法目标则本次效果结束，不储存余压 |
| 信息反馈 | 电击结束后分出一条细小余弧，明确飞向新的未感电目标 |

## 7. 通用桥接神器

### 7.1 U-A1 交替协议

一句话：任何两种不同主动技能轮流释放都能形成费用回流，鼓励组合操作而不是只常驻一个技能。

一句话：燃烧清除、感电结算或冰冻结束后留下一个短暂接力点，让下一种技能优先追向刚刚完成结算的战场位置。

| 防递归 | 余波不能触发直接命中监听、元素状态、罗盘或其他复制效果 |
| 异常处理 | 位置附近无合法目标时不消费罗盘点；超时后自然消失 |
| 信息反馈 | 结算位置出现元素颜色罗盘，下一技能锁定时拉出一条瞄准线 |

一句话：主动技能成功把敌人推离墙体或打断危险动作时，把防守价值即时转换成少量技能费用。

## 10. 神器候选生成与随机下限

### 10.1 立即有效判定

神器满足以下任一条件时，视为“当前立即有效”：

- 玩家已持有能产生其必需事件的主动技能或进化；
- 玩家已持有主动技能，且该技能当前可见的进化路线能够产生其必需事件；
- 神器本身不要求元素输入，例如 U-A1；
- 神器本身能够从当前技能的基础效果启动完整循环。

不把“未来可能抽到某个尚未持有技能”视为立即有效。

### 10.2 三选一下限

- 每次神器三选一至少包含一件当前立即有效神器。
- 其余选项可以用于转向或预埋，但三件不能全部依赖玩家尚未拥有的输入。
- 精确权重、尾随机和连续未命中保底留到概率/数值阶段。
- 开局获得神器时，以初始持有主动技能及其当前可见进化路线判断有效性；若仍无元素输入，必须包含通用神器。

### 10.3 互斥与降权

- 获得 I-A1 后，I-A3、I-A4、I-A5 从本局后续候选池移除；已获得任一冰冻系神器时，I-A1 不再进入候选。
- 没有任何寒冷来源时，I-A1/I-A2 不进入立即有效位。
- 没有任何感电来源时，E-A1～E-A5 不进入立即有效位。
- 没有燃烧清除能力时，F-A2/F-A3 降为转向候选，不占立即有效保底位。
- 已有费用返还神器时，可以继续获得另一件费用神器，但必须展示共享返还上限，且生成权重应降低 `[待验证]`。

## 11. 规则优先级与冲突处理

| 冲突 | 处理规则 |
|---|---|
| I-A1 与基础寒冷满层冰冻 | I-A1 替换冰冻转化；仍发出寒冷满层，不发出进入冰冻 |
| I-A1 与 I2/I3 神器 | 本局互斥，不允许共同获得 |
| E-A1 与基础电击后清除 | E-A1 替换真实清除，但保留“感电结算完成/本应清除”监听事件 |
| E-A1 与 E-A5 | E-A5 监听“本应清除”，因此仍可转接余压；原目标感电保留 |
| E-A1 与 E-A2 | 群星导线照常逐目标结算；各目标按 E-A1 保留感电并进入自身触发间隔 |
| F-A2 与同次燃爆 | 同一技能和同一根事件不能点燃余烬，必须由后续不同主动技能接力 |
| 多个费用返还神器 | 分别判断资格，最后合并返还；合计值受单次释放总返还上限约束且低于实际费用 |
| 多个控制神器 | 同一目标先结算主动技能控制，再结算一次神器附加控制；位移和动作控制分别受 Boss 抗性与目标间隔限制 |

## 12. 赛季扩展接口

后续赛季可以沿以下方向增加神器，但不得用纯倍率上位替代首发神器：

### 12.1 深化既有出口

- 火：新的燃烧转场、区域留痕、清除后重建方式；
- 冰：新的寒冷追击轨迹、碎冰几何、冰锚连线方式；
- 电：新的放电邻接、目标排序、危险动作读取方式。

### 12.2 跨属性桥接

- 只读取公开事件：燃烧满层/清除、寒冷存在、进入/结束冰冻、感电结算、成功控制、技能交替；
- 不新增热量、寒能、电荷槽等内部资源；
- 不要求固定 A 技能＋B 技能配方；
- 桥接神器本身必须对至少一侧输入有独立价值。

### 12.3 新规则改写

- 可以改变状态结算步骤、目标选择和战场几何；
- 必须写明基础规则、替代规则、事件迁移、既有状态迁移和互斥关系；
- 不能覆盖寒冷固定减速等 `FORBIDDEN` 禁区；
- 新规则不能让首发基础 Build 失去成立能力。

### 12.4 神器池增长

- 赛季新增神器后仍保持每次三选一至少一件立即有效；
- 若候选池增长导致有效命中率下降，应通过条件池、属性尾随机或赛季轮换解决；
- 是否采用轮换池、预装池或动态筛选尚未确定，留到长期投放设计；
- 不因池子扩大把旧神器改成纯数值低配版。

## 13. D3 数值接口

每件神器进入 D3 时必须补齐：

- 触发伤害或追击修正；
- 状态施加层数与状态上限；
- 目标数、搜索距离、范围、持续时间；
- 每目标间隔、每次释放次数、单位时间次数；
- 费用返还值和单次释放总返还上限；
- Boss 控制系数；
- 视觉实体数与真实结算数；
- 普通局、胡牌局和大胡局中的出现概率与有效命中率；
- 两件、三件神器同时生效时的总价值预算。

其中 I-A1 的寒冷上限 `10 层` 当前仍为机制候选，需在 D3 根据追击收益、叠层速度和信息表现验证后确认。

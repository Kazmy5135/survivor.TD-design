## 1. 目标与边界

本节点只确定 MVP 需要哪些流派、每条流派向玩家提供什么操作与战斗结果，以及它们如何共享公共输入、适配有限槽位并为后续赛季留接口。

本节点定义：

- 火、冰、电三元素的总身份；
- 每个元素的三条候选流派；
- 每条流派的公共输入、价值出口、操作、视觉和战场职责；
- 乞丐版、基础完全体与胡牌方向；
- 属性抗性以外的行为强弱场景；
- 同属性共享、跨属性桥接和赛季扩展位置；
- 4 槽与 6 槽阶段的成立边界。

本节点不定义：

- 最终武器名单或某种表现一定入池；
- 具体进化、被动和神器名称；
- 具体伤害、费用、层数、时间或倍率；
- 具体敌人原型；
- 属性反应和固定双属性配方。

## 2. 生效的 D1 基础语义

D2.1 必须遵守以下 Accepted 规则：

- 燃烧是可叠层持续伤害，持续伤害不可暴击；
- 基础感电在目标受到直接伤害时触发一次电击，电击后清除全部感电；
- 同一次命中若触发并清除感电，不能在事件尾部立即重挂感电；
- “电击后保留感电”只能作为未来神器的显式规则改变器，不能作为电流派基础成立条件；
- 寒冷从 1 层到满层前 1 层始终提供同一个固定减速；
- 寒冷层数可以作为伤害或其他收益的连续计算输入，但不能形成不可见的中间阈值，也不能提高减速；
- 寒冷满层后全部转化为冰冻；冰冻及免疫期间不能获得寒冷；
- 任何主动、进化、被动、神器或成长均不得让寒冷减速随层数提高；
- 神器可以显式改变寒冷满层转化、施加层数或层数上限，但不能覆盖“寒冷减速不随层数提高”的禁区；
- 层数阈值只使用“满层”，消耗状态时清除全部当前层数；
- 同一状态的层不记录玩家不可见的技能来源差异。

## 3. MVP 流派生态总图

D2.1 当前提案为火、冰、电各三条流派，共九条；每个元素采用“两条可承担主输出的路线＋一条副方向路线”。

| 编号 | 元素 | 流派暂名 | 定位 | 公共输入 | 价值出口 | 核心操作动词 |
|---|---|---|---|---|---|---|
| F1 | 火 | 燎原灼烧 | 主输出 | 燃烧存在与持续覆盖 | 大范围持续伤害 | 覆盖、维持、刷新 |
| F2 | 火 | 灼烧爆破 | 主输出 | 当前全部燃烧层 | 全量清除后的集中爆发 | 积累、选择、引爆 |
| F3 | 火 | 余火供能 | 副方向 | 燃烧满层、燃烧击杀或燃烧结算事件 | 技能费用恢复或费用效率 | 覆盖、回收、接力 |
| I1 | 冰 | 极寒追击 | 主输出 | 目标处于寒冷 | 长时间维持寒冷后的直接伤害追击 | 标记、维持、持续追击 |
| I2 | 冰 | 碎冰爆发 | 主输出 | 寒冷满层形成的冰冻窗口 | 提前结束冰冻换取爆发 | 叠满、冻结、碎裂 |
| I3 | 冰 | 寒潮封锁 | 副方向 | 寒冷满层形成的冰冻锚点 | 冰冻时产生群体回推、动作打断与阵型重排 | 冻结、回拒、清线 |
| E1 | 电 | 弧链放电 | 主输出 | 多目标已施加感电 | 范围直接命中触发多目标电击并清除 | 播种、扫击、连锁 |
| E2 | 电 | 过载聚焦 | 主输出 | 高价值目标的感电层数 | 单次直接命中触发高层电击并清除 | 充能、等待、放电 |
| E3 | 电 | 脉冲干扰 | 副方向 | 威胁目标已施加感电 | 下一次直接命中的电击附带短动作干扰 | 标记、判断、打断 |

### 3.1 三元素总身份

| 元素 | 主要伤害节奏 | 副方向 | 玩家主要决策 | 主要视觉语言 |
|---|---|---|---|---|
| 火 | 状态随时间保留，选择继续维持或全部兑现 | 供能与跨技能续航 | 保留燃烧、引爆燃烧或回收燃烧事件 | 连续火焰、燃烧覆盖、集中爆燃、能量回流 |
| 冰 | 用固定减速争取时间，满层进入短冰冻窗口 | 冰冻回拒与防线空间重建 | 长期维持寒冷、提前碎冰，或利用冰冻锚点推回敌群 | 霜层标记、持续追击、冻结碎裂、寒潮回推 |
| 电 | 先施加感电，再由下一次直接命中放电并清除 | 短暂动作干扰 | 给谁充能、何时命中放电、放电覆盖多少目标 | 感电标记、一次脉冲、电弧跳转、集中放电 |

## 4. 火属性流派

### 4.1 F1 燎原灼烧

一句话身份：持续扩大燃烧覆盖，让更多目标在更长时间内承受完整燃烧伤害。

| 维度 | 定义 |
|---|---|
| 主要方向 | 群体持续伤害 |
| 公共输入 | 目标已燃烧、燃烧持续时间、状态覆盖目标数 |
| 价值出口 | 燃烧持续伤害和覆盖效率 |
| 操作体验 | 选择能覆盖更多目标或延长覆盖的释放时机，避免在即将死亡的目标上浪费持续时间 |
| 视觉体验 | 大片敌群维持统一燃烧表现，死亡与战场空缺逐步扩散 |
| 战场职责 | 群体、持续、长波次清场 |
| 乞丐版 | 一个能够稳定施加燃烧的核心职责即可建立持续伤害方向 |
| 基础完全体 | 挂燃烧职责＋扩大覆盖或延长有效时间的不同职责 |
| 胡牌方向 | 2～3 个职责共同提高覆盖、持续结算和燃烧全局强度，但不依赖唯一组件 |
| 行为强项 | 密集持续刷怪、长波次、目标存活时间足够的敌群 |
| 行为弱项 | 极短命目标、状态清除、必须立刻处理的短威胁窗口 |
| 共享位置 | 与 F2、F3 共享燃烧施加、覆盖和满层事件 |
| 赛季接口 | 传播方式、区域留存、燃烧击杀后的新出口；不修改燃烧基础定义 |

### 4.2 F2 灼烧爆破

一句话身份：先积累燃烧，再选择合适目标和时机清除全部燃烧，换取一次集中爆发。

| 维度 | 定义 |
|---|---|
| 主要方向 | 群体或精英携群爆发 |
| 公共输入 | 目标当前全部燃烧层、满层状态、附近目标关系 |
| 价值出口 | 按实际清除层数结算的触发伤害或范围爆发 |
| 操作体验 | 判断继续等待更高积累，还是提前引爆解决当前威胁 |
| 视觉体验 | 燃烧状态明确消失，随后出现一次集中爆燃；同事件不重挂燃烧 |
| 战场职责 | 爆发、收口、精英携群处理 |
| 乞丐版 | 一个能建立燃烧并具备低频全量引爆接口的核心职责可以形成最小循环 |
| 基础完全体 | 稳定挂燃烧职责＋独立全量引爆职责 |
| 胡牌方向 | 提高积累效率、引爆覆盖或清除层数价值，但不允许部分消耗 |
| 行为强项 | 高生命目标周围有小怪、集中清场窗口、短时间必须处理的威胁 |
| 行为弱项 | 燃烧积累不足、目标过度分散、频繁状态清除 |
| 共享位置 | 使用 F1 的覆盖输入，也可为 F3 提供清除事件 |
| 赛季接口 | 爆燃传播、清除后的战场留存或与其他公开事件桥接 |

### 4.3 F3 余火供能

一句话身份：把燃烧覆盖产生的公开事件转化为技能费用收益，为其他技能和混合流派提供续航。

| 维度 | 定义 |
|---|---|
| 主要方向 | 副方向、供能、跨技能接力 |
| 公共输入 | 燃烧满层、燃烧击杀、燃烧被清除等公开事件 |
| 价值出口 | 技能费用恢复、费用节省或释放节奏改善 |
| 操作体验 | 维持足够燃烧覆盖，再把获得的费用投入其他主动技能 |
| 视觉体验 | 燃烧事件后出现有限次数的能量回流反馈，不生成“热量”等新资源 |
| 战场职责 | 供能、混合 Build 支持、长波次续航 |
| 乞丐版 | 一个燃烧职责＋一个非唯一的供能接口即可为当前技能组提供收益 |
| 基础完全体 | 稳定燃烧覆盖职责＋能把公开事件转成费用的职责 |
| 胡牌方向 | 多技能共享费用收益并改善释放顺序，但必须有单位时间收益上限 |
| 行为强项 | 大量持续刷怪、能稳定产生满层或击杀事件的场景 |
| 行为弱项 | 单体且事件频率低、状态清除、费用已经长期溢出的技能组 |
| 共享位置 | 可辅助冰、电或无属性主流派，不要求火承担主要伤害 |
| 赛季接口 | 新的公开事件回收方式、跨技能费用窗口；禁止内部计数资源 |

## 5. 冰属性流派

### 5.1 I1 极寒追击

一句话身份：长时间维持目标身上的寒冷与固定减速，并持续用冰属性直接命中追击；目标进入冰冻后不再满足极寒追击条件。

| 维度 | 定义 |
|---|---|
| 主要方向 | 单体/穿透型持续直接伤害 |
| 公共输入 | 目标处于寒冷；冰冻不算作有效目标条件 |
| 价值出口 | 长时间寒冷覆盖下的追踪、穿透和直接伤害；追击输出可以随当前寒冷层数连续提高，但不存在中间层数阈值 |
| 操作体验 | 优先延长和刷新寒冷，以较低的单次施加量维持长期减速，再持续命中寒冷目标；进入冰冻会暂时中断追击收益 |
| 视觉体验 | 统一霜层标记作为目标条件，主要反馈来自持续冰击和碎屑命中 |
| 战场职责 | 高生命目标追击、快速目标处理、持续直接输出 |
| 乞丐版 | 一个能长时间刷新寒冷且对寒冷目标有基础追击能力的核心职责即可成立 |
| 基础完全体 | 寒冷持续维持职责＋只追击寒冷目标的直接伤害职责 |
| 胡牌方向 | 提高寒冷维持、追击命中率和按当前层数连续计算的直接伤害，不读取中间层数作为离散阈值 |
| 行为强项 | 高生命、高速、需要持续追踪的目标 |
| 行为弱项 | 状态频繁清除、目标极短命、过快满层导致频繁进入冰冻/免疫空窗 |
| 共享位置 | 与 I3 共享寒冷覆盖，与 I2 共享冰冻窗口 |
| 神器规则改变候选 | 核心神器可使寒冷满层不再转化冰冻、让施加寒冷的攻击额外增加层数，并把寒冷上限改为 10 层；保留下来的高层寒冷继续放大追击输出，但减速仍为固定值；具体组合留到 D2.2 |
| 赛季接口 | 状态目标追踪、穿透、返回，或把长期保留寒冷发展成新的上位追击方式 |

### 5.2 I2 碎冰爆发

一句话身份：通过寒冷满层制造冰冻，再选择提前结束冰冻，以失去剩余控制时间为代价换取一次碎冰爆发。

| 维度 | 定义 |
|---|---|
| 主要方向 | 控制窗口转换为直接或触发爆发 |
| 公共输入 | 寒冷满层事件、目标当前冰冻状态 |
| 价值出口 | 提前结束冰冻后的碎冰伤害和有限范围扩散 |
| 操作体验 | 判断保留控制保护防线，还是立即碎冰解决威胁 |
| 视觉体验 | 冰冻后出现一次清晰碎裂，冰冻表现同步结束 |
| 战场职责 | 爆发、短威胁窗口、精英携群处理 |
| 乞丐版 | 一个能推进寒冷满层且具备基础碎冰接口的职责可形成较慢循环 |
| 基础完全体 | 稳定叠寒冷职责＋独立碎冰收口职责 |
| 胡牌方向 | 提高冰冻建立效率、碎冰范围或结束控制后的输出，但不能形成永久控制 |
| 行为强项 | 可被冻结的精英携群、冲锋或蓄力前的短窗口 |
| 行为弱项 | 控制免疫、寒冷无法积累、过早碎冰导致控制与伤害同时不足 |
| 保底边界 | 即使目标不能冰冻，相关技能仍保留自身基础直接伤害；碎冰额外收益可以失效 |
| 赛季接口 | 碎裂传播、冻结结束事件桥接、以部分控制时间换不同战场结果 |

### 5.3 I3 寒潮封锁

一句话身份：把满层冰冻当作战场锚点；冰冻发生时释放一次寒潮回拒，把附近敌人推离墙体、打断动作并重新拉开防线空间。

| 维度 | 定义 |
|---|---|
| 主要方向 | 副方向、控场、覆盖辅助 |
| 公共输入 | 寒冷满层事件、当前冰冻目标及其附近敌群 |
| 价值出口 | 冰冻目标周围的一次群体击退、动作中断与阵型打散；固定减速只负责前置争取时间 |
| 操作体验 | 选择敌群最接近墙体或阵型最密集的时机制造冰冻锚点，再用寒潮回拒整体推开威胁 |
| 视觉体验 | 一个目标被冻结成为清晰锚点，随后寒潮从锚点向外扩散，敌群整体向远离墙体的方向回退 |
| 战场职责 | 群体回推、防线空间重建、冲锋打断、混合 Build 控场 |
| 乞丐版 | 一个能稳定推进冰冻并在冰冻时产生小范围回拒的核心职责即可提供独立控制价值 |
| 基础完全体 | 寒冷/冰冻建立职责＋扩大寒潮回拒范围或提高阵型打散能力的职责 |
| 胡牌方向 | 多个冰冻锚点错峰回拒、扩大安全输出窗口或与范围伤害接力；必须设置单位时间触发与击退上限 |
| 行为强项 | 已靠近墙体的密集敌群、冲锋阵型、精英携群和容易形成拥堵的路线 |
| 行为弱项 | 远程原地攻击、目标极度分散、击退免疫或高控制抗性目标 |
| 共享位置 | 为火持续覆盖重新拉开结算时间，为电群体播种重整目标位置，为重击技能创造安全窗口 |
| 赛季接口 | 冰冻锚点数量、回拒后的区域留痕、敌群聚拢或改道等更具战场结构感的控制方式 |

寒潮回拒属于一次位移/控制事件，不是更强的减速：

- 不提高寒冷固定减速值；
- 每次冰冻最多产生一次寒潮回拒；
- 同一目标与单位时间内必须存在触发上限；
- Boss 可以缩短冰冻并降低或免疫击退，但不能让相关技能的基础伤害失效；
- 大规模敌群中的视觉可以裁剪，真实结算目标数仍必须有明确上限。

## 6. 电属性流派

电的基础操作循环统一为：

```text
施加感电
→ 选择下一次直接命中
→ 按触发前层数产生电击
→ 清除全部感电
→ 重新寻找施加窗口
```

神器未来可以显式把“清除”改为“保留”，但九条 MVP 基础流派不得依赖该神器才能成立。

### 6.1 E1 弧链放电

一句话身份：先给一群目标播种感电，再用范围直接命中同时触发多目标电击和有限连锁，完成群体放电。

| 维度 | 定义 |
|---|---|
| 主要方向 | 群体响应伤害 |
| 公共输入 | 多个目标已感电、目标之间的距离或邻接关系 |
| 价值出口 | 范围直接命中触发的多目标电击与有限目标连锁 |
| 操作体验 | 先扩大感电目标数，再选择覆盖最多目标的直接命中时机 |
| 视觉体验 | 感电标记后出现一次有限数量的清晰电弧跳转，随后标记消失 |
| 战场职责 | 密集群体清场、精英周围小怪处理 |
| 乞丐版 | 一个能交替施加感电和直接命中的核心职责可以形成较慢的播种—放电循环 |
| 基础完全体 | 多目标感电播种职责＋独立范围直接命中/连锁职责 |
| 胡牌方向 | 提高播种覆盖、放电有效目标和有限连锁效率，但必须保留目标数与表现上限 |
| 行为强项 | 密集移动群、精英携群、目标关系稳定的区域 |
| 行为弱项 | 远距离分散目标、孤立 Boss、频繁目标切换导致播种浪费 |
| 共享位置 | 与 E2、E3 共享感电播种；可读取冰减速创造的聚集时间 |
| 赛季接口 | 连锁规则、邻接关系、神器提供的感电保留；不得递归触发感电 |

### 6.2 E2 过载聚焦

一句话身份：在不提前直接命中的情况下给高价值目标积累感电，再用一次明确的直接命中释放高层电击并清除。

| 维度 | 定义 |
|---|---|
| 主要方向 | 单体/Boss 集中爆发 |
| 公共输入 | 高价值目标当前感电层数、满层状态 |
| 价值出口 | 一次直接命中触发的高层电击伤害 |
| 操作体验 | 控制充能与直接命中的先后，避免过早命中导致低层放电 |
| 视觉体验 | 目标感电表现由普通进入满层阶段，随后一次集中脉冲并清除 |
| 战场职责 | 精英、Boss、单体短窗口处理 |
| 乞丐版 | 一个具有“施加阶段→直接命中阶段”的分段核心职责可以完成基础过载 |
| 基础完全体 | 非直接伤害的感电施加职责＋独立单体直接命中收口职责 |
| 胡牌方向 | 提高单次施加层数、充能安全性和放电收益；不能使用小于满层的固定阈值 |
| 行为强项 | 高生命、目标稳定、拥有明确输出窗口的精英或 Boss |
| 行为弱项 | 大量短命小怪、目标频繁消失、无法避免的低频提前命中 |
| 共享位置 | 播种职责可与 E1 共用，放电窗口可与 E3 的威胁判断共用 |
| 赛季接口 | 不同充能载体、满层可见反馈、神器保留感电形成持续过载上位路线 |

### 6.3 E3 脉冲干扰

一句话身份：先给即将发动危险动作的目标施加感电，再用下一次直接命中触发电击和一次短暂动作干扰。

| 维度 | 定义 |
|---|---|
| 主要方向 | 副方向、威胁打断、短控制窗口 |
| 公共输入 | 威胁目标已感电、目标正在准备攻击/冲锋/蓄力 |
| 价值出口 | 电击附带的一次动作延迟、中断或短硬直 |
| 操作体验 | 识别高威胁目标的动作窗口，再决定何时直接命中放电 |
| 视觉体验 | 一次集中脉冲中断动作；不形成持续电弧或永久控制 |
| 战场职责 | 威胁管理、防线容错、混合 Build 支持 |
| 乞丐版 | 一个能施加感电并在后续直接命中提供基础干扰的职责即可产生功能价值 |
| 基础完全体 | 稳定标记威胁目标职责＋可控时机的直接命中职责 |
| 胡牌方向 | 提高目标选择、干扰窗口可靠性和跨属性输出窗口；必须保留同目标间隔 |
| 行为强项 | 冲锋、蓄力、远程重击或短暂护盾等可观察动作 |
| 行为弱项 | 没有关键动作的持续压线敌人、干扰免疫或高抗性 Boss |
| 共享位置 | 为火爆破、冰碎裂和无属性重击创造安全命中窗口 |
| 赛季接口 | 新的可观察动作读取、干扰后的跨技能窗口；禁止永久控制 |

## 7. 完成度与槽位适配

### 7.1 统一完成度

| 完成度 | 结构 | 体验要求 |
|---|---|---|
| 乞丐版 | 1 个核心职责，或 1 个技能内部的两个阶段 | 能表达该流派的输入—出口，不因缺少唯一神器完全失效 |
| 基础完全体 | 2 个职责不同的主动技能或进化方向 | 一个建立公共输入，一个负责利用、兑现或放大 |
| 胡牌 | 2～3 个相关技能/进化＋支持被动＋1～2 个有效神器 | 效率或操作方式出现明显跃迁，但仍遵守触发和表现上限 |
| 大胡方向 | 2～3 个职责完整吻合，并获得跨技能或规则改变支持 | 接近理论上限，不能形成无限资源、永久控制或递归触发 |

### 7.2 4 槽阶段

```text
2 个主流派职责
＋0～1 个辅助属性职责
＋1～2 个通用/自由职责
```

- 每条流派必须由 1 个核心职责形成乞丐版；
- 2 个不同职责形成基础完全体；
- 第 3 个同流派职责只能提高胡牌上限，不能成为基本闭环硬门槛。

### 7.3 6 槽阶段

```text
2～3 个主流派职责
＋1～2 个辅助/跨属性职责
＋至少 1 个自由职责
```

- 新槽位优先增加混合构筑和容错；
- 不把成熟 Build 改写成 5～6 个固定组件；
- 后续技能可以提供新的表达或桥接方式，不能只是首发技能的纯倍率上位替代。

## 8. 流派关系与桥接接口

### 8.1 同属性共享

| 元素 | 三流派共享输入 | 分流位置 |
|---|---|---|
| 火 | 燃烧施加、覆盖、满层、击杀和清除事件 | 保留持续伤害、全量引爆、费用回收 |
| 冰 | 寒冷存在、满层、冰冻及冰冻结束 | 长期寒冷追击、碎冰爆发、冰冻锚点回拒 |
| 电 | 感电施加、直接命中、电击与全量清除 | 群体放电、单体过载、动作干扰 |

### 8.2 跨属性允许读取的公开接口

- 目标带有燃烧、寒冷或感电；
- 目标进入某状态满层；
- 燃烧被全部清除；
- 目标进入或结束冰冻；
- 感电触发电击并被全部清除；
- 目标处于固定减速；
- 与上一次不同的主动技能造成直接命中；
- 一个技能为另一个技能创造费用、覆盖或安全输出窗口。

跨属性桥接必须遵守：

- 不新增热量、电荷、寒能等内部计数资源；
- 不要求唯一 A 技能＋B 技能固定配方；
- 不让一种属性的触发结果递归触发自身；
- 辅助、控制、供能和覆盖价值必须计入总预算；
- 本节点只保留接口，具体神器在 D2.2 设计。

## 9. 行为型强弱场景

| 行为场景 | 优势流派 | 受限流派 | 原因 |
|---|---|---|---|
| 密集连续刷怪 | F1、F3、E1 | E2 | 覆盖、事件频率和目标邻接关系更有价值 |
| 精英携群 | F2、I2、E1 | 纯单体追击 | 集中爆发或有限扩散能同时处理主次目标 |
| 孤立高生命目标 | I1、E2 | E1、F3 | 状态追击和单体充能可以集中价值 |
| 高速推进 | I1、I3、E3 | 慢启动 F1 | 长期固定减速、冰冻回拒和短干扰争取处理时间 |
| 冲锋/蓄力短窗口 | F2、I2、E3 | 纯持续覆盖 | 爆发、冻结或动作干扰能及时响应 |
| 目标高度分散 | I1、E2、局部 F1 | E1、小范围 F2 | 定点追击比邻接连锁更稳定 |
| 状态周期清除 | 直接伤害占比高的 I1、E2 | F1、F2、F3 | 长时间维持和积累更容易损失 |
| 控制抗性 | F1、F2、E1、E2 | I3、E3 | 回拒和动作干扰降效，但相关技能基础伤害仍需保留 |
| 远程原地攻击 | F1、F2、E2 | I3 | 固定减速对不移动目标价值低 |

## 10. 赛季扩展边界

后续赛季只允许从以下方向扩展：

1. 深化现有出口，例如新的覆盖、传播、追击或收口方式；
2. 桥接两个已有出口，但不要求固定配方；
3. 为保留技能/进化建立新价值出口；
4. 使用神器显式改变某条规则，例如让电击后保留感电；
5. 增加新的攻击几何或时间形态，而不反复修改元素基础状态。

长期禁区：

- 寒冷减速不得随层数提高；
- 基础感电不得默认变成电击后保留；
- 不新增只用于搬运数值的元素内部资源；
- 不用同条件、同效果、纯倍率更高的内容淘汰首发路线；
- 不让上位路线破坏低配 Build 的正常成立能力。

允许的冰系神器脑洞包括取消寒冷满层冰冻、增加每次寒冷施加层数或修改寒冷层数上限；这些变化可以让极寒追击的输出随当前层数连续成长，但不能把层数重新变成逐层减速，也不能增加不可见的中间阈值。

## 11. D2.2 内容需求接口

D2.1 Accepted 后，D2.2 需要从本流派生态反推：

- 每条流派需要哪些职责，而不是先填武器名；
- 哪些职责可以由同一个武器类型或进化跨流派复用；
- 攻击几何、时间形态、属性进入时机和第一次进化定位；
- 哪些是输入位、利用位、收口位、供能位、控制位和自由位；
- 哪些神器支持公共输入、桥接两个出口或只负责单一收口；
- 感电保留神器如何单件有效，同时不成为电基础流派的硬前提；
- 极寒追击的规则改变神器如何取消冰冻转化、增加寒冷施加层数或把层数上限改为候选 10 层，并把保留层数转成连续追击收益，同时保持固定减速；
- 寒潮封锁需要哪些冰冻锚点、群体回拒、动作打断和阵型重排职责；
- 首发内容覆盖哪些需求，哪些接口保留给赛季扩展。

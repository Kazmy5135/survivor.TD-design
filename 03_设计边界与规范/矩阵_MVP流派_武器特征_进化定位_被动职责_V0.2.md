## 1. 本轮重拆原因

V0.1 把“多数基础武器无属性、第一次进化再获得元素”写成默认结构，限制了武器本体的元素身份，也没有在同一步拆出进化所依赖的被动职责。

本轮按设计负责人修订：

- 武器基础状态不一定无属性；
- 可以存在基础即为火属性、后续第一次和第二次进化始终保持火属性的武器；
- 固定元素武器的不同路线应在同一元素内部改变攻击方式和流派职责，而不是为了复用强行跨属性；
- 武器分解必须同时给出对应被动的大类效果，至少明确“每个被动加什么、解锁哪些进化方向”；
- 暂不填写被动具体名称、最终倍率和等级数值。

原 V0.1 的 Accepted 结论已撤回；本文以 `Review` 状态重新审核。

## 2. 武器属性模型

### 2.1 固定元素武器

基础形态已经拥有元素属性和对应状态能力。

- 基础、三条第一次进化及各自第二次进化均保持同一元素；
- 路线差异来自攻击几何、时间形态、状态用途和流派出口；
- 例如火属性武器可以分别深化燃烧覆盖、灼烧爆破输入和余火供能，而不必出现冰或电路线；
- 固定元素武器可以成为玩家开局就能识别的属性入口。

### 2.2 可转属性武器

基础形态为无属性，定向进化把同一武器骨架转成不同元素用途。

- 默认路线保留无属性并提供合格基础战力；
- 两条定向路线可以分别获得不同元素；
- 一旦选择第一次进化，后续中间 Perk 与第二次进化必须沿该属性继续；
- 不允许第二次进化临时切换到另一元素。

### 2.3 功能型无属性武器

基础价值来自防护、位移、锁敌或费用处理，不依赖元素伤害。

- 默认路线继续强化通用功能；
- 定向路线可以获得元素响应，也可以继续深化非元素功能；
- 即使没有匹配元素 Build，也必须保留独立的防线或资源价值。

### 2.4 属性继承规则

- “基础属性”与“进化获得属性”都必须在武器定义中显式填写；
- 固定元素武器不会因为默认进化而失去自身元素；
- 一个进化路线可以从无属性变为单一元素，但单个伤害事件仍只能有一个主属性；
- 多段技能若包含不同属性，必须拆成独立伤害事件；
- 元素属性不等于必须只服务一个流派，同一火武器可以在火的三条流派之间分化。

## 3. 武器特征拆解字段

每个武器槽至少拆解以下特征：

| 特征 | 必须回答的问题 |
|---|---|
| 基础元素 | 无属性、固定火、固定冰或固定电？ |
| 基础动作 | 玩家释放后最主要的攻击行为是什么？ |
| 攻击几何 | 扇形、弹道、直线、弹射、波前、留场或墙体区域？ |
| 时间形态 | 瞬时、多段、持续、延迟、往返或双拍？ |
| 锁敌规则 | 优先最近、密集区、高价值目标、寒冷目标、感电目标或墙前威胁？ |
| 状态能力 | 基础是否施加元素状态？进化怎样改变状态用途？ |
| 战场职责 | 输入、利用、结算、控制、供能、防护或自由位？ |
| 基础强弱项 | 不依赖进化时擅长和不擅长什么？ |
| 三条第一次进化 | 默认路线与两条被动定向路线分别改变什么？ |
| 第二次进化边界 | 怎样强化第一次路线，哪些新职责不得加入？ |
| 对应被动 | 哪个被动解锁定向路线，该被动本身增加什么？ |

## 4. 首发8个武器特征槽

以下名称为特征工作名，不是最终武器名。D2.3 可以更换表现和命名，但必须保留已确认的基础元素、攻击骨架和职责覆盖。

### 4.1 W1 烈焰喷流

基础身份：固定火属性的近中距离持续喷流武器；基础、全部第一次进化和对应第二次进化均保持火属性。

| 维度 | 定义 |
|---|---|
| 基础动作 | 向自动选定的敌群持续喷出多段火焰 |
| 攻击几何 | 扇形或窄扇形 |
| 时间形态 | 一次释放生成有限时长的高频多段喷流 |
| 基础状态 | 每段直接伤害后施加少量燃烧 |
| 基础强项 | 近墙群体覆盖、高频挂燃烧、持续压制 |
| 基础弱项 | 远端目标、分散目标、孤立 Boss 的命中效率 |
| 默认进化 | `广域焰幕`：保持火属性，扩大扇面与转场覆盖，主服务 F1 燎原灼烧 |
| 定向进化 A | `聚束高温`：需要 P6 精准增幅；保持火属性，缩窄喷流并提高单体挂层效率，服务 F2 的燃烧积累 |
| 定向进化 B | `滞留火带`：需要 P4 持续增幅；保持火属性，喷流结束后留下短时燃烧区域，服务 F1 并增加 F3 的燃烧事件 |
| 第二次进化边界 | 继续强化扇面、聚束或留场；不得变成全量燃烧清除器，不切换冰/电属性 |

### 4.2 W2 炽焰弹道

基础身份：固定火属性的低频爆炸弹道武器；基础、全部第一次进化和对应第二次进化均保持火属性。

| 维度 | 定义 |
|---|---|
| 基础动作 | 自动锁定高密度区域或高价值目标，发射火属性弹体并产生局部爆炸 |
| 攻击几何 | 单体弹道＋落点圆形范围 |
| 时间形态 | 有飞行时间的低频单次爆发 |
| 基础状态 | 爆炸直接伤害后对命中目标施加燃烧 |
| 基础强项 | 精英携群、密集区域、明显命中反馈 |
| 基础弱项 | 高频小怪、目标移动导致的落点损失、持续覆盖不足 |
| 默认进化 | `爆燃弹头`：保持火属性；未燃烧目标获得燃烧，已燃烧目标全量清除并燃爆，主服务 F2 |
| 定向进化 A | `分裂燃弹`：需要 P5 范围增幅；保持火属性，弹体分裂或扩大爆区，主服务 F1 并辅助 F2 群体输入 |
| 定向进化 B | `余火回收`：需要 P7 费用循环；保持火属性，命中燃烧目标或燃烧击杀后产生有限费用回流，主服务 F3 |
| 第二次进化边界 | 分别强化燃爆、分裂覆盖或费用回收；费用返还受总上限约束，不切换冰/电属性 |

### 4.3 W3 寒霜轨刃

基础身份：固定冰属性的直线穿透与回返武器；基础、全部第一次进化和对应第二次进化均保持冰属性。

| 维度 | 定义 |
|---|---|
| 基础动作 | 沿高价值目标方向穿透敌群，命中后施加寒冷 |
| 攻击几何 | 窄直线、有限穿透，可在进化后回返 |
| 时间形态 | 单次去程或去程＋回程 |
| 基础状态 | 直接伤害后施加寒冷；寒冷固定减速不随层数提高 |
| 基础强项 | 纵向队列、快速目标、高价值目标追踪 |
| 基础弱项 | 横向分散敌群、近墙大面积包围 |
| 默认进化 | `霜迹追猎`：保持冰属性，优先锁定寒冷目标并获得追击收益；冰冻目标不计入，主服务 I1 |
| 定向进化 A | `裂晶回程`：需要 P6 精准增幅；保持冰属性，回程命中冰冻目标时提前结束冰冻并碎裂，主服务 I2 |
| 定向进化 B | `冰锚折线`：需要 P8 防线控制；保持冰属性，命中形成冰冻时从锚点产生一次折线回拒，主服务 I3 |
| 第二次进化边界 | 强化追猎、碎晶回程或锚点折线；不得让寒冷减速随层数提高，不切换火/电属性 |

### 4.4 W4 雷鸣脉冲

基础身份：固定电属性的双拍脉冲武器；基础、全部第一次进化和对应第二次进化均保持电属性。

| 维度 | 定义 |
|---|---|
| 基础动作 | 第一拍播种感电，第二拍产生直接命中并触发电击 |
| 攻击几何 | 目标脉冲、短链或小范围波 |
| 时间形态 | 同一次释放内两个可区分节拍 |
| 基础状态 | 第一拍只施加感电；第二拍直接命中，按基础规则电击后清除 |
| 基础强项 | 单技能形成完整感电循环、节拍清晰、容易理解 |
| 基础弱项 | 两拍间目标死亡、目标切换、对大量短命怪可能溢出 |
| 默认进化 | `弧链脉冲`：保持电属性，扩大第一拍播种与第二拍群体放电目标，主服务 E1 |
| 定向进化 A | `聚焦过载`：需要 P6 精准增幅；保持电属性，两拍集中同一高价值目标并提高满层放电收益，主服务 E2 |
| 定向进化 B | `断招脉冲`：需要 P8 防线控制；保持电属性，优先锁定准备危险动作的目标并在放电时中断，主服务 E3 |
| 第二次进化边界 | 强化群链、单体聚焦或断招可靠性；没有神器时仍必须电击后清除，不切换火/冰属性 |

### 4.5 W5 重型落击

基础身份：无属性重击武器，通过定向进化转为火或冰结算器。

| 维度 | 定义 |
|---|---|
| 基础动作 | 自动锁定密集区或高价值目标，延迟后造成一次重型范围直接伤害 |
| 攻击几何 | 单体落点＋圆形或纵向冲击范围 |
| 时间形态 | 明显前摇后的单次爆发 |
| 基础状态 | 无元素状态 |
| 基础强项 | 通用爆发、精英携群、默认路线稳定 |
| 基础弱项 | 前摇落空、高频覆盖不足、费用较重的风险 |
| 默认进化 | `重压扩爆`：保持无属性，强化落点范围、命中稳定性和自由位爆发 |
| 定向进化 A | `燃烧落弹`：需要 P1 火焰增幅；转为火属性，未燃烧时施加燃烧、已燃烧时全量燃爆，服务 F2 |
| 定向进化 B | `寒晶坠击`：需要 P2 冰霜增幅；转为冰属性，未冰冻时施加寒冷、冰冻时提前碎冰，服务 I2 |
| 第二次进化边界 | 沿无属性重压、火焰燃爆或冰晶碎裂继续强化；不得在第二次进化跨线改属性 |

### 4.6 W6 弹射飞刃

基础身份：无属性有限弹射武器，通过定向进化转为火焰传播或电弧群链。

| 维度 | 定义 |
|---|---|
| 基础动作 | 从初始目标向附近尚未命中的目标有限弹射 |
| 攻击几何 | 目标间邻接网络 |
| 时间形态 | 一次释放形成短弹射序列 |
| 基础状态 | 无元素状态 |
| 基础强项 | 密集群体、精英携群、目标关系稳定区域 |
| 基础弱项 | 孤立 Boss、远距离分散敌群、目标过少 |
| 默认进化 | `多重折跃`：保持无属性，增加合法弹射与目标去重，作为群体自由位 |
| 定向进化 A | `火种跃迁`：需要 P1 火焰增幅；转为火属性，在未燃烧目标间传播燃烧，服务 F1/F3 |
| 定向进化 B | `电弧导链`：需要 P3 电势增幅；转为电属性，一次释放播种、下一次释放依次放电并清除，服务 E1 |
| 第二次进化边界 | 强化折跃、火种传播或电弧导链；禁止传播或电击递归启动自身 |

### 4.7 W7 防线波炮

基础身份：无属性波前控制武器，通过定向进化转为冰冻锚点或感电断招。

| 维度 | 定义 |
|---|---|
| 基础动作 | 从墙体向敌群推出一次横向或带状波前 |
| 攻击几何 | 横向波、向上推进带或锚点圆环 |
| 时间形态 | 一次释放一轮推进，同目标每轮最多结算一次 |
| 基础状态 | 无属性直接伤害＋基础轻击退 |
| 基础强项 | 墙前密集敌群、冲锋阵型、防线空间整理 |
| 基础弱项 | 远程原地敌人、分散目标、击退免疫 Boss |
| 默认进化 | `宽幅回推`：保持无属性，强化波宽和通用击退，作为控制自由位 |
| 定向进化 A | `寒潮锚波`：需要 P2 冰霜增幅；转为冰属性，施加寒冷并在本次形成冰冻时产生回拒，服务 I3，也可为 I1 提供寒冷输入 |
| 定向进化 B | `跳闸波`：需要 P3 电势增幅；转为电属性，对感电目标放电并中断危险动作，服务 E3 |
| 第二次进化边界 | 强化回推、冰锚回拒或电击断招；不得形成永久控制或重复命中循环 |

### 4.8 W8 守护壁垒

基础身份：功能型无属性防护武器；默认路线强化墙体防护，定向路线获得火焰反击或费用循环。

| 维度 | 定义 |
|---|---|
| 基础动作 | 在墙体前建立一个有限时长防护窗口 |
| 攻击几何 | 固定墙前防护带或全墙体响应 |
| 时间形态 | 一次释放生成有限时长效果，窗口内事件有上限 |
| 基础状态 | 无属性防护，不施加元素状态 |
| 基础强项 | 墙体容错、近墙威胁、任何 Build 都能使用 |
| 基础弱项 | 直接清场能力低、敌人未接近墙体时价值降低 |
| 默认进化 | `坚壁响应`：保持无属性，提高拦截、减伤或墙体保护，作为稳定防护位 |
| 定向进化 A | `焰返壁垒`：需要 P1 火焰增幅；获得火属性反击，对首次进入或攻击墙体的敌人施加燃烧，服务 F3 并辅助 F1 |
| 定向进化 B | `循环护盾`：需要 P7 费用循环；保持无属性，把成功拦截或防护事件转成有限费用回流，服务 F3 与混合 Build |
| 第二次进化边界 | 强化防护、火焰反击或费用循环；费用返还低于实际支付费用且受总上限限制 |

## 5. 首发8个被动职责

本轮只确定被动“加什么”和“解锁哪些进化”，不确定最终名称、倍率和完整 Lv.1/Lv.2 文案。

| ID | 被动职责工作名 | Lv.1 主要增加 | Lv.2 继续增加 | 主要解锁方向 | 明确边界 |
|---|---|---|---|---|---|
| P1 | 火焰增幅 | 火属性直接伤害与燃烧伤害 | 继续提高火输出或燃烧施加效率 | W5 火焰燃爆、W6 火种传播、W8 火焰反击 | 不改变燃烧基础清除语义 |
| P2 | 冰霜增幅 | 冰属性直接伤害与寒冷持续/施加效率 | 继续提高冰输出或冰冻建立效率 | W5 寒晶碎冰、W7 寒潮锚波 | 不让寒冷减速随层数提高 |
| P3 | 电势增幅 | 电属性直接伤害与感电施加效率 | 继续提高电击利用或播种效率 | W6 电弧导链、W7 跳闸波 | 不让基础感电在电击后保留 |
| P4 | 持续增幅 | 持续区域、持续伤害或状态持续时间 | 继续提高覆盖稳定性与转场能力 | W1 滞留火带 | 不独立制造新的元素状态 |
| P5 | 范围增幅 | 技能范围、目标数或弹体覆盖 | 继续提高范围稳定性或目标上限 | W2 分裂燃弹 | 真实结算目标仍受性能上限 |
| P6 | 精准增幅 | 单体直接伤害、穿透或高价值目标锁定 | 继续提高精英/Boss 收口与命中稳定性 | W1 聚束高温、W3 裂晶回程、W4 聚焦过载 | 不提高持续伤害暴击能力 |
| P7 | 费用循环 | 技能费用自然恢复、返还效率或交替释放收益 | 继续提高费用容错和循环稳定性 | W2 余火回收、W8 循环护盾 | 所有返还合计低于实际支付费用 |

### 5.1 被动与进化关系

- 默认进化始终不要求被动，必须具备合格基础战力；
- 定向进化只要求对应被动达到 Lv.1；
- 对应被动达到 Lv.2 后，才满足该路线第二次进化条件；
- 一个被动可以解锁多个武器路线，避免“一武器一专属被动”；
- 被动基础效果在没有匹配进化时也必须有效；
- 元素被动提高对应元素能力，但不能擅自改写 D1 基础状态规则；
- 功能被动可以支持固定元素武器，例如火焰喷流可以由持续、范围或精准被动进一步分化。

### 5.2 长线新增4个被动职责

长线从8个被动扩展到12个时，新增被动优先补足费用结构、轨迹玩法和近墙响应，不重复制作P1～P8的纯倍率上位版。

| ID | 被动职责工作名 | 主要增加 | 长线武器接口 | 明确边界 |

| P9 | 费用容量 | 共享技能费用池上限、技能能量点恢复效率或高费用技能容错 | 延迟布置、蓄能阵地、重费用武器 | 不直接返还超过实际支付的费用 |
| P10 | 自动循环 | 自动重复请求的稳定性、长时效果衔接或不同技能循环效率 | 移动战线、巡航区域、长时持续技能 | 不把按次费用改回持续耗能 |

## 1. 一句话定义

玩家使用火焰喷射器对怪群持续叠加燃烧，再用导弹一次清除目标的全部燃烧，并按清除层数造成即时爆破；单技能可以独立播种或利用燃烧，双技能齐全时形成“叠层—清除—重新叠层”的清晰循环。

## 2. 设计目标

- 燃烧层数提供数值成长空间，但不要求玩家读取任何敌人的精确层数。
- 火焰喷射器负责高频范围叠层，导弹负责全量清除和范围收口。
- 同一敌人身上的所有燃烧层使用统一伤害，不记录火焰喷射器或导弹来源。
- 技能局部 Perk 只修改本技能的直接伤害、范围、费用、挂层速度或满层响应。
- 消耗燃烧后必须让燃烧特效真实消失，同一导弹事件不重新挂燃烧。
- 单技能形成乞丐版，双技能形成胡牌，进化、被动和 2～3 件神器吻合时形成大胡。

## 3. Build 身份卡

| 维度 | 定义 |
|---|---|
| 属性/流派 | 火 / 燃烧叠层与全量爆破 |
| 核心关键词 | 燃烧、满层、清除全部燃烧、范围伤害、击杀转移 |
| 核心主动技能 | 火焰喷射器、导弹 |
| 主要输出 | 火焰喷射器直接伤害、统一燃烧持续伤害、导弹燃爆伤害 |
| 强项 | 怪群、精英携带小怪、持续推进波次 |
| 弱项 | 极短命目标、火抗目标、无法维持火焰覆盖的场景 |

| 成型速度 | 单技能可启动，双技能明显成型 |

## 4. 战场信息与燃烧规则

### 4.1 信息边界

- 战场约有 400～500 个敌人；
- 不在敌人头顶显示燃烧数字层数；
- 不显示燃烧层来自哪个技能；
- 同一敌人的全部燃烧只使用一个统一特效；
- 满层如需反馈，只允许强化同一个燃烧特效或在触发时给出反馈，不增加逐层图标。

### 4.2 燃烧

| 字段 | 当前定义 | 状态 |
|---|---|---|
| 层数上限 | 10 层 | 测试候选 |
| 再次施加 | 增加来源规定层数，并刷新整组持续时间 | 已确认 |
| 每层效果 | 所有来源统一，按固定间隔造成一次火属性持续伤害 | 已确认 |
| 暴击 | 燃烧持续伤害不可暴击 | 已确认 |
| 超出上限 | 超出部分丢弃，不转换资源 | 已确认 |
| 中间层数条件 | 禁止使用 | 已确认 |
| 消耗方式 | 一次清除目标当前全部燃烧 | 已确认 |
| 消耗后状态 | 本次事件不重新施加，燃烧特效消失 | 已确认 |
| 持续时间 | 未定 | 待验证 |
| 跳伤间隔/每层伤害 | 未定 | 待验证 |

### 4.3 允许玩家理解的状态

```text
未燃烧
→ 已燃烧
→ 燃烧满层
→ 被导弹清除后回到未燃烧
```

玩家不需要区分 3 层、5 层或 7 层。

## 5. 结算顺序

### 5.1 火焰喷射器命中

```text
结算火焰喷射器直接伤害
→ 若目标未满层，施加本次固定燃烧层数并刷新持续时间
→ 若结算后达到满层，产生“燃烧满层”事件
```

### 5.2 导弹命中未燃烧目标

```text
结算导弹直接或范围伤害
→ 施加导弹自带的固定燃烧层数
→ 本目标结算结束
```

### 5.3 导弹命中已燃烧目标且拥有燃烧弹头进化

```text
结算导弹直接或范围伤害
→ 记录目标当前全部燃烧层数
→ 清除全部燃烧，燃烧特效消失
→ 按实际清除层数结算燃爆伤害
→ 本目标结算结束，不重新施加燃烧
```

同一枚导弹对每个受击目标分别判断“未燃烧播种”或“已燃烧清除”，但整次释放的额外爆破伤害存在总上限。

## 6. 主动技能

### 6.1 火焰喷射器

#### 基础形态

- 发射一条火柱；
- 火柱经过的敌人每 0.5 秒受到一次火属性直接伤害；
- 每次伤害确定性施加 1 层燃烧；
- 同一敌人每 0.5 秒最多由该技能获得一次燃烧施加；
- 火焰喷射器直接伤害可以按全局规则暴击，燃烧持续伤害不能暴击。

燃烧施加层数为候选值，基础技能表现来自 F0 事实，挂层规则来自本 Build 设计推导。

#### 本 Build 定向进化：高温灼流

进化条件候选：火焰喷射器达到第一次进化节点，且 `火势强化 Lv.1`。

第一次进化效果：

> 火焰喷射器每次命中未满层燃烧目标时，施加的燃烧由 1 层提高为 2 层。目标满层后不再继续添加层数。

路线中间 Perk 候选：

1. **宽幅喷口**：提高火柱宽度，不增加同一目标的命中频率。
2. **高压火柱**：提高火焰喷射器的直接伤害，不改变燃烧每层伤害。
3. **节流阀门**：降低火焰喷射器单次释放的技能费用，不改变释放和命中频率。

第二次进化：白热焚烧

> 火焰喷射器命中满层燃烧目标时，额外造成一次小额火属性直接伤害；同一目标每 1 秒最多触发一次，不消耗燃烧。

设计作用：单独持有火焰喷射器时，也能完成“稳定叠层—进入满层—获得额外直接伤害”的乞丐版循环。所有路线 Perk 都能在不区分燃烧来源的前提下表达。

### 6.2 导弹

#### 基础形态

- 发射一枚导弹，对命中目标造成一次火属性直接伤害；
- 对命中目标周围敌人造成一次火属性范围伤害；
- 没有燃烧弹头进化时，每个被本次导弹造成伤害的敌人确定性获得 2 层燃烧；
- 主目标默认只结算一次燃烧施加，是否同时受到范围伤害由技能基础规格另行确认。

燃烧施加层数为候选值，基础技能表现来自 F0 事实，挂层规则来自本 Build 设计推导。

#### 本 Build 定向进化：燃烧弹头

进化条件候选：导弹达到第一次进化节点，且 `火势强化 Lv.1`。

第一次进化效果：

> 导弹命中未燃烧敌人时，正常施加基础燃烧。导弹命中已燃烧敌人时，清除该敌人的全部燃烧；每清除 1 层，追加一定倍率的火属性燃爆伤害。本次清除结算后不重新施加燃烧。

路线中间 Perk 候选：

1. **高爆装药**：提高导弹范围，但不重复命中同一敌人。
2. **燃爆增压**：提高导弹每清除 1 层燃烧获得的燃爆伤害，不改变燃烧本身的每层伤害。
3. **精确引信**：导弹清除满层燃烧时，本次燃爆伤害获得额外倍率。

第二次进化：彻底燃爆

> 导弹清除满层燃烧时，提高本次燃爆的每层伤害，并扩大该目标产生的燃爆范围；仍然清除全部燃烧，且本次事件不重新施加燃烧。

设计作用：单独持有导弹时，第一次释放负责播种，下一次释放负责清除并燃爆，此后重复播种与燃爆；与火焰喷射器配合时，可以跳过导弹自身较慢的播种过程。

## 7. 被动技能候选

### 火势强化

> 类型：火属性专属被动。本轮只占用 1 个火属性被动名额。

| 等级 | 效果 | 进化接口 |
|---|---|---|
| Lv.1 | 提高火属性技能的直接伤害，不修改燃烧每层伤害 | 解锁火焰喷射器“高温灼流”和导弹“燃烧弹头”路线 |
| Lv.2 | 技能每次成功施加燃烧时，额外施加 1 层 | 满足对应路线第二次进化条件 |

数值状态：全部待验证。

其他可用但不由本 Build 独占的被动接口：

- 范围强化：提高火柱宽度或导弹爆炸范围；
- 能量回复/能耗方向：支持火焰喷射器持续覆盖与导弹主动收口；
- Boss/单体方向：提高导弹对单一高生命目标的基础直接伤害。

## 8. 神器候选

以下神器系统地位等价，每件都提供独立价值；编号只用于引用，不代表品质或获取顺序。

### F-A1：火上浇油

> 火属性技能向已经处于燃烧状态且尚未满层的敌人施加燃烧时，额外施加 1 层。单次伤害事件只追加一次。

- 单件价值：所有持续攻击或重复释放的火技能更快达到满层；
- 支持火焰喷射器：第二次命中开始提高叠层速度；
- 支持导弹：没有燃烧弹头进化时，连续导弹提高叠层速度；
- 边界：不响应燃烧持续伤害，不因本神器追加的层数再次触发自身。

### F-A2：爆燃震荡

> 一个技能事件清除敌人的全部燃烧时，在该敌人位置追加一次小范围火属性爆炸；爆炸基础强度随本次实际清除层数提高。

- 单件价值：强化所有具有“清除全部燃烧”接口的技能或进化；
- 支持导弹：燃烧弹头每次成功清除燃烧时提供额外范围收口；
- 边界：每个目标每次技能事件最多触发一次；爆炸不施加燃烧，也不能再次触发爆燃震荡。

### F-A3：余烬蔓延

> 燃烧敌人死亡时，使距离最近的一个未满层敌人获得与死亡目标相同的当前燃烧层数，并刷新燃烧持续时间。

- 单件价值：把即将随死亡消失的燃烧转移到下一个目标；
- 支持火焰喷射器：怪群持续进入火柱时形成燃烧接力；
- 支持导弹：燃爆击杀后为下一目标播种；
- 边界：只影响一个目标；超过上限的层数丢弃；转移不触发自身；没有有效目标时效果结束。

### F-A4：稳定燃烧

> 所有来源施加的燃烧，其每层持续伤害统一提高一定比例。

- 单件价值：直接强化当前所有燃烧来源，不依赖特定技能或另一件神器；
- 作用域：火焰喷射器、导弹和未来火技能施加的燃烧统一生效；
- 边界：不记录来源，不让不同技能施加的燃烧产生不同每层伤害；
- 数值状态：倍率待验证。

`稳定燃烧` 是全局 Debuff 强化，因此放在神器层级，不再出现在火焰喷射器的技能路线 Perk 中。

## 9. 核心循环

```text
火焰喷射器范围命中
→ 多个敌人稳定获得统一燃烧层数
→ 部分敌人进入满层燃烧
→ 导弹命中已燃烧敌人
→ 清除各目标全部燃烧并按层数燃爆
→ 燃烧特效消失
→ 后续火焰喷射器或下一次导弹重新播种
```

最小闭环不要求全部组件：

- 火焰喷射器可以独立叠层并在满层后获得额外收益；
- 导弹可以在未燃烧目标上播种，下一发自行清除并燃爆；
- 任意一件匹配神器都能独立强化叠层、燃爆、转移或全局燃烧伤害。

## 10. Build 完成度

### 10.1 乞丐版 A：持续灼烧

```text
火焰喷射器
＋高温灼流路线或火势强化
＋F-A1 火上浇油 / F-A3 余烬蔓延 / F-A4 稳定燃烧任一
```

- 玩法：持续覆盖怪群，使燃烧进入满层并依靠持续伤害与白热焚烧清场；
- 不需要导弹；
- 不读取精确层数。

### 10.2 乞丐版 B：交替燃爆

```text
导弹
＋燃烧弹头路线
＋F-A2 爆燃震荡 / F-A4 稳定燃烧任一
```

- 玩法：对未燃烧目标播种，对已燃烧目标清除全部燃烧并燃爆；
- 不需要火焰喷射器；
- 每次清除后燃烧真实消失，下一次导弹重新播种。

### 10.3 胡牌版：喷射叠层＋导弹清除

```text
火焰喷射器的高温灼流路线
＋导弹的燃烧弹头路线
＋火势强化
＋任意两件匹配神器
```

- 火焰喷射器解决导弹单独使用时需要隔次播种的问题；
- 导弹解决火焰喷射器只有持续输出、缺少主动爆发出口的问题；
- 两件神器可以分别强化叠层、燃爆、击杀接力或全局燃烧伤害，不要求固定组合。

### 10.4 大胡/完全体

```text
火焰喷射器第二次进化“白热焚烧”
＋导弹第二次进化“彻底燃爆”
＋火势强化 Lv.2
＋四件候选神器中的任意 2～3 件
```

推荐但不唯一的三种完全体偏向：

| 偏向 | 神器组合 | 结果 |
|---|---|---|
| 怪群接力 | 火上浇油＋爆燃震荡＋余烬蔓延 | 快速叠层、清除爆炸、击杀后转移 |
| 持续燃烧 | 火上浇油＋余烬蔓延＋稳定燃烧 | 强化覆盖、转场和 DOT，不依赖频繁导弹 |
| 精英/Boss | 火上浇油＋爆燃震荡＋稳定燃烧 | 更快满层，并把全部层数转成高价值燃爆 |

玩家只需判断哪些区域已被持续点燃、是否出现满层反馈，以及当前敌群是否值得导弹收口，不需要识别某个敌人的具体层数。

## 11. 非唯一解校验

| 缺少组件 | 替代路径 | 是否还能运行 |
|---|---|---|
| 没有导弹 | 火焰喷射器满层收益＋火上浇油/余烬蔓延/稳定燃烧 | 是，偏持续清群 |
| 没有火焰喷射器 | 导弹在播种与全量燃爆之间交替 | 是，偏低频爆发 |
| 没有火势强化 | 使用默认进化维持基础战力，神器仍能强化燃烧公共行为 | 是，但上限降低 |
| 没有爆燃震荡 | 导弹自身仍按清除的全部层数造成燃爆 | 是 |
| 没有余烬蔓延 | 对后续目标重新播种 | 是，转场效率降低 |
| 没有火上浇油 | 基础技能仍确定性挂层 | 是，满层较慢 |
| 没有稳定燃烧 | 叠层和燃爆机制仍完整 | 是，持续伤害较低 |

结论：不存在“获得 A 后只能等待 B”的唯一启动组件或唯一神器组合。

## 12. 数值拆解接口

### 12.1 核心公式

```text
燃烧单次跳伤 = 统一每层燃烧基础伤害 × 当前燃烧层数 × 全局燃烧伤害修正

导弹单目标燃爆伤害 =
min（本次清除全部燃烧层数 × 每层燃爆倍率 × 导弹基础伤害，单目标燃爆上限）

本次导弹额外燃爆总伤害 =
min（所有受击目标的燃爆伤害之和，本次释放燃爆总上限）
```

### 12.2 变量

| 变量 | 含义 | 单位 | 候选/范围 | 状态 |
|---|---|---|---|---|
| BurnMaxStack | 燃烧层数上限 | 层 | 10 | 测试候选 |
| BurnDuration | 燃烧持续时间 | 秒 | 未定 | 待验证 |
| BurnTickInterval | 燃烧跳伤间隔 | 秒 | 未定 | 待验证 |
| BurnDamagePerStack | 所有来源共用的每层燃烧跳伤 | 伤害 | 未定 | 待验证 |

## 13. 验收标准

### 13.1 功能验收

- [ ] 火焰喷射器每 0.5 秒对同一目标最多执行一次基础燃烧施加。
- [ ] 同一敌人身上的全部燃烧共享统一每层伤害和持续时间，不保存技能来源差异。
- [ ] 燃烧增加层数时刷新整组持续时间，超过上限的部分被丢弃。
- [ ] 燃烧持续伤害不能暴击。
- [ ] 导弹命中已燃烧目标时清除全部燃烧，不保留部分层数。
- [ ] 导弹清除燃烧后，本次事件不重新施加燃烧，状态特效实际消失。
- [ ] 导弹命中未燃烧目标时才能执行本次基础播种分支。
- [ ] 神器追加攻击和层数转移不会递归触发自身。

### 13.2 信息与体验验收

- [ ] 文案不出现要求玩家识别的 3 层、5 层、7 层等中间阈值。
- [ ] 玩家不读取精确层数也能理解喷射器负责点燃、导弹负责清除燃爆。
- [ ] `稳定燃烧` 统一影响全部燃烧来源，不出现在单技能路线 Perk 中。
- [ ] 燃烧被导弹清除时，统一燃烧特效与燃爆反馈同步完成。
- [ ] 在 400～500 个敌人的战场中不增加来源图标、逐层数字或逐层独立特效。
- [ ] 只有一个核心技能时仍能获得完整、可感知的成长。
- [ ] 双技能组合后，清怪方式或爆发节奏出现明显变化。

### 13.3 平衡验收

- [ ] 低随机结果可以依靠默认进化和单件神器维持基础战力。
- [ ] 导弹不能通过一次 AOE 无上限读取全场燃烧并秒杀 Boss。
- [ ] 火焰喷射器不会因为叠层能力同时成为最强直接伤害技能。
- [ ] 全量清除不会让燃烧持续伤害长期失去结算时间。
- [ ] 大胡相对乞丐版有明显效率提升，但不达到无需操作的程度。

## 14. 验收标准

### 14.1 功能验收

- [ ] 事件、状态、顺序和防递归可被程序实现。

### 14.2 信息与体验验收

- [ ] 玩家无需读取不可见信息即可理解和操作 Build。

### 14.3 数值验收

- [ ] AOE、持续伤害与引爆没有重复按满价值计入。
- [ ] 高完成度明显变强，但没有秒杀、无限循环或无需操作。

### 14.4 长线验收

- [ ] 当前 Build 没有吞掉全部技能进化和神器扩展空间。
